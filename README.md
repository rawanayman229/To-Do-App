To Do List App

A modern Kanban-style Todo application built with React, TypeScript, and Drag & Drop functionality. Users can create, organize, and manage tasks across different workflow stages while filtering and searching tasks efficiently.

## 📸 Features

### ✅ Task Management
- Create new tasks
- Delete tasks
- Store tasks in Local Storage
- Persistent data after page refresh

### 📋 Kanban Board
- Todo column
- In Progress column
- Done column

### 🎯 Drag & Drop
- Move tasks between columns
- Reorder tasks inside the same column
- Smooth drag-and-drop experience using `@hello-pangea/dnd`

### 🏷 Labels
- Assign labels to tasks

### 🔍 Search & Filtering
- Search tasks by title
- Filter tasks by label
- Instant filtering and search results

### 📱 Responsive Design
- Desktop friendly
- Tablet support
- Mobile responsive layout
